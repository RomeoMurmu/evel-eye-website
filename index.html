01

Clone the Repository

Download Evil-Eye from GitHub to your local machine.

git clone https://github.com/RGSecurityTeam/evil-eye.git cd evil-eye
02

Install Dependencies

Install required packages. Evil-Eye will auto-install most on first run, but pre-installing is recommended.

sudo apt update sudo apt install -y php php-cgi curl wget jq # lolcat and cloudflared are auto-installed on first run
brew install php curl wget jq gem install lolcat
03

Make Executable & Run

Give execution permission and launch Evil-Eye.

chmod +x evel.sh bash evel.sh
04

Port Conflict Fix (Parrot OS)

If you have GoPhish or another tool using port 3333, change Evil-Eye's port to avoid conflict.

# Check what's using port 3333 sudo ss -tlnp | grep 3333 # Kill conflicting process or change Evil-Eye's port sed -i 's/127.0.0.1:3333/127.0.0.1:4444/g' lib/tunnel.sh

04   Usage Guide

How to Use

Step-by-Step Workflow

  • Run bash evel.sh ‚Äî tool checks internet, verifies all dependencies.
  • Select a template from the main menu (1‚Äì5).
  • Fill in any required input (e.g. victim name, YouTube URL, message file).
  • Select tunnel option ‚Äî Cloudflared (public link) or Localhost.
  • Copy the generated Cloudflare link and send it to the target.
  • Terminal will show when the link is opened and images are received.
  • Captured images saved to captured_imgs/ folder.
  • Press Ctrl+C to terminate. Images auto-move to output folder.

Captured Data

  • captured_imgs/ ‚Äî JPEG images from target's camera, named with timestamp.
  • captured_ip.txt ‚Äî Target IP address and user-agent logged here.
  • Camera permission prompt appears on the target's browser ‚Äî once accepted, capture begins silently every 1.5 seconds.
  • Images captured at full native camera resolution (up to 4K).
  • All data is stored locally ‚Äî nothing sent to any third-party server.

Tunnel Options

  • Cloudflared ‚Äî Generates a public trycloudflare.com link. Works from any network. Recommended.
  • Localhost ‚Äî Runs on 127.0.0.1:3333 only. Useful for local testing on the same device or LAN.

Tips & Notes

  • Camera access requires HTTPS ‚Äî Cloudflared provides this automatically.
  • On Parrot OS, ensure php-cgi is installed as a fallback server.
  • GoPhish uses port 3333 by default ‚Äî stop it before running or change Evil-Eye's port.
  • The link only works while the terminal session is active. Closing the terminal stops the server.
